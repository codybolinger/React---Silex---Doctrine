# React---Silex---Doctrine
A Sample CRUD app using React front end, Silex PHP backend and Doctrine Postgres DB

Start PHP Server - 
```
cd /silex-PHP-server
php -S localhost:8000
```

Start React Server -
```
cd my-crud-app
npm start
```
